# Von Pasion — Portfolio

Personal portfolio website. Solutions Architect, Tech Consultant, Builder.

## Stack

- **Framework:** Next.js 16 (App Router, static export)
- **Styling:** Tailwind CSS v4
- **Animation:** Framer Motion
- **Language:** TypeScript
- **Deploy:** Vercel

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
```

Static output goes to `out/`.

## Content

All content lives in `src/data/`:

- `projects.ts` — featured work
- `timeline.ts` — career journey
- `principles.ts` — core principles
- `social.ts` — contact links
- `utils.ts` — shared helpers (years of experience)

Edit the data files, push, Vercel auto-deploys.
